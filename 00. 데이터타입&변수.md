# Python 기초: 데이터 타입과 변수

## 1. 데이터 타입 (Data Types)

### 숫자 데이터 타입
- **int(정수)**: 소수점이 없는 숫자
- **float(실수)**: 소수점이 있는 숫자
  - `print(10.)` -> `10.0`으로 인식
- **연산 규칙**: 
  - 정수/정수 나눗셈은 언제나 **실수** (`4/2` -> `2.0`)
  - 실수와의 연산 결과는 언제나 **실수**

### 연산자
- `//`: 몫 구하기 (소수점 버림)
- `%`: 나머지 구하기
- `**`: 제곱 (`2**4` -> `16`)

### 문자열 타입 (str)
- 만드는 법: `" "`, `' '`, `""" """`, `''' '''`
- **연산**:
  - `+`: 문자열 연결
  - `*`: 문자열 반복
  - `len()`: 문자 개수 구하기

### 논리 타입 (bool)
- `True`, `False` (대문자 주의)
- 비교 연산자(`>`, `<`, `==`, `!=`)의 결과값

---

## 2. 변수 (Variable) 및 형변환

### 변수 사용
- `변수명 = 값` 형태로 대입
- 파이썬은 변수 값 교체가 간편함: `a, b = b, a`

### 타입 변환 함수
- `int()`, `float()`, `str()`, `bool()`
- **주의**: 문자열과 숫자를 더할 때는 반드시 `str()`로 형변환 필요

### 복합 대입 연산자
- `a += 10` (기존 값에 10을 더함)
- `b /= 2` (기존 값을 2로 나눔)
